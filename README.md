# Некоторые проекты из области Data Science:

##  Разработка системы глубокого обучения для задачи диагностики расстройства аутистического спектра (РАС)

Расстройства аутистического спектра представляют собой группу психических 
расстройств, которая характеризуется нарушениями в социальном взаимодействии, 
коммуникации и поведении. Раннее выявление РАС у детей является критически 
важным для своевременного вмешательства и улучшения качества жизни. 
Целью данной работы является разработка модели глубокого обучения, способной 
эффективно выявлять признаки РАС по фотографиям детей. 

Задачи, поставленные в рамках работы, включают:
1. Подготовка датасета и рассмотрение возможных проблем.
2. Разработка архитектуры модели глубокого обучения для детекции РАС.
3. Обучение модели на наборе данных.
4. Анализ эффективности полученной модели, оценка её точности.
5. Валидация модели: анализ её практической применимости в медицинской 
диагностике

Наблюдения:
1. Предобученные модели показали хорошие результаты.
2. DenseNet-121 достигла наилучшей точности (0.84), но показала низкую скорость.
3. ResNet-18 не сильно уступает ей в качестве, зато обучалась почти в 4 раза 
быстрее.
4. Ранняя остановка срабатывала для всех моделей, что позволило избежать 
переобучения.
5. Улучшенная CNN с BatchNorm и Dropout превзошла базовую на 3.7%, что 
подтверждает важность методов регуляризации.

##  Разработка системы глубокого обучения для диагностики болезни Альцгеймера на основе МРТ-снимков мозга

Болезнь Альцгеймера — хроническое дегенеративное заболевание головного мозга, 
характеризующееся резким снижением интеллектуальных способностей вплоть до 
тотального распада интеллекта и психической деятельности.
Целью данной работы является разработка модели глубокого обучения, способной 
выявить ранние признаки болезни Альцгеймера на основании снимков МРТ головного 
мозга.

Архитектура:
· 3 сверточных блока (Conv2d + ELU + MaxPool2d)
· 2 полносвязных слоя (предварительно выполнено разворачивание в вектор)
· Dropout для регуляризации (каждый нейрон выключается с вероятностью 0.3)
Модель обучилась за 489 секунд и достигла точности 0.9932 на тестовом множестве.
